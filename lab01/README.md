# Project description

This is a demostration of the usage of Hardhat on ETH. It includes a smart contract that anyone can transfer money into it, but only the owner (set when deploy) can withdraw.

# Setup instructions

> [!NOTE]
> I use pnpm as package manager, if npm does not work, try to switch to pnpm

install dependency

```bash
$ npm install
```

# Test instructions

```bash
$ npm run hardhat test
```
